Set up Typora + Picgo + Github notes environment

***Typora***: A markdown app that can easily write markdown notes

***Picgo***:  Every picture you copied into typora, picgo will send it to github (after a little configuration)

***Github***: Where we host our images. We will create a repo like that:

## Step 1: Create a new Github Repository

We use Github to host our images. Maybe make it public so other people can see your images. The name, description doesn't matter.

## Step 2: Set up GitHub token

Click your Github profile -> then setting:

Then, give this token a name. I set expiration to no expiration since I'm lazy to config it again soon.

Also make sure you check repo.This is the ***only*** thing that we checked. Then, click generate token!
